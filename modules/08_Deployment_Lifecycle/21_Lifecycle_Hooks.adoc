== Lifecycle Hooks
:noaudio:

.Overview

* `Recreate` and `Rolling` strategies support lifecycle hooks
** Allow behavior to be injected into deployment process at predefined points

* `pre` lifecycle hook:
+
----
"pre": {
  "failurePolicy": "Abort",
  "execNewPod": {}
}
----

** `execNewPod` is pod-based lifecycle hook
* Every hook has `failurePolicy`
