== OpenShift Enterprise Population

* Core set of image streams defines:
** Images you can use to build applications:
*** Node.js
*** Perl
*** PHP
*** Python
*** Ruby

** Images for databases:
*** MongoDB
*** MySQL
*** PostgreSQL

[NOTE]
The install utility installs these image streams and others automatically.
