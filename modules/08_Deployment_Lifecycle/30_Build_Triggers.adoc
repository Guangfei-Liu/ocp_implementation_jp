== Build Triggers



* To define build with strategy that consumes image stream:
+
----
{
  "strategy": {
    "type": "Source",
    "sourceStrategy": {
      "from": {
        "kind": "ImageStreamTag",
        "name": "ruby-20-rhel7:latest"
      },
    }
  }
}
----
+
** `sourceStrategy` definition consumes `latest` tag of image stream named
 `ruby-20-rhel7` located in this namespace
