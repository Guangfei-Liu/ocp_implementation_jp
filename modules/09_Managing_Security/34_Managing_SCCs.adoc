== Managing SCCs


.Creating SCCs: Part 1

* To create SCC, first define SCC in JSON or YAML file
* SCC object definition:
+
----
kind: SecurityContextConstraints
apiVersion: v1
metadata:
  name: scc-admin
allowPrivilegedContainer: true
runAsUser:
  type: RunAsAny
seLinuxContext:
  type: RunAsAny
users:
- my-admin-user
----
